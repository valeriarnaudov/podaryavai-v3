# 🎁 Podaryavai V3

**Podaryavai** е модерно уеб приложение, създадено да улесни процеса на планиране, откриване и закупуване на перфектния подарък. Версия 3.0 надгражда основата с изкуствен интелект, автоматизирана геймификация и пълен административен контрол.

---

## ✨ Ключови Функционалности

### 🤖 AI Giftinder (Персонализиран Асистент)
*   **Swipe интерфейс (Tinder-style)** за бързо разглеждане на подаръци.
*   **Истинска Персонализация:** Генерира ежедневни, уникални идеи чрез **OpenAI (GPT-4o)**, базирани на историята на нещата, които потребителят е харесвал.
*   **Скалируемост:** Броят на дневните идеи и качеството им зависят от абонаментния план (от 3 за Free до 15 за Business).

### 💎 Karma Rewards (Геймификация)
*   Точкова система (Karma Points), която награждава потребителите за тяхната активност (напр. добавяне на контакти, покана на приятели).
*   **Вграден Магазин (Karma Shop)** за директно харчене на точките:
    *   `Golden Aura` – анимирана златна рамка около аватара.
    *   `Free Delivery` – премахва таксата при поръчка на подарък чрез Concierge.
    *   `VIP Giftinder` – отключва премиум селекция от ръчно подбрани подаръци.
    *   `Karma Multiplier` – х2 бууст на всички спечелени точки за 24 часа.

### 🗂️ Интелигентен Wishlist
*   Разделя и организира запазените подаръци в две секции:
    *   **My Requests:** Ръчно добавени желания (със собствени линкове и цени).
    *   **From AI Giftinder:** Идеите, които са запазени при Swipe-ване.

### 🛡️ Административен Панел (Admin Dashboard)
*   Пълен контрол над екосистемата за потребители с роля `is_admin`.
*   **Управление на Потребители:** Търсене, забрана (Ban), даване на права. Моментално нулиране на дневния AI лимит на даден потребител.
*   **Глобални Настройки:** Редактиране на параметри (напр. колко точки дава един реферал) директно от UI-а.
*   **Управление на Подаръци:** CRUD операции за глобалната VIP база от подаръци.

### 🛎️ Автоматични Нотификации
*   Система от фонови процеси (Supabase Edge Functions + pg_cron), която ежедневно проверява за предстоящи събития (Рождени дни, Имени дни) и изпраща навременни email напомняния чрез Resend API.

---

## 🛠️ Технологичен Стек

*   **Frontend:** React (Vite), TypeScript, Tailwind CSS, Framer Motion, Lucide Icons, React Router DOM.
*   **Backend & Cloud:** [Supabase](https://supabase.com/) (PostgreSQL Database, Authentication, Edge Functions, Cron Jobs).
*   **AI Инеграция:** OpenAI API (`gpt-4o`).
*   **Имейли:** Resend API.
*   **Плащания:** Stripe (Concierge checkout integration).

## 🚀 Инсталация и Локално Стартиране (Local Development)

### 1. Клониране на хранилището
```bash
git clone https://github.com/valeriarnaudov/podaryavai-v3.git
cd podaryavai-v3
```

### 2. Инсталиране на зависимостите
```bash
npm install
```

### 3. Environment Variables
Създайте файл `.env` в главната директория със следното съдържание:
```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 4. Стартиране на Development Сървъра
```bash
npm run dev
```
Приложението ще се стартира на `http://localhost:5173`.
